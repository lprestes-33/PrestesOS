# Como usar o Kit de Governança

Copie estes arquivos para a raiz do repositório PrestesOS.

## Comando sugerido no Termux

```bash
unzip PrestesOS_Governance_Kit.zip -d ~/PrestesOS
cd ~/PrestesOS
git add AGENTS.md .ai MASTER_SPEC.md ARCHITECTURE.md ROADMAP.md TASKS.md CHANGELOG.md README_GOVERNANCE.md
git commit -m "docs: adiciona kit de governança para agentes"
```

## Prompt curto para Codex

```text
Leia AGENTS.md e siga a governança do repositório. Identifique a próxima tarefa prioritária no ROADMAP/TASKS, implemente com testes, atualize documentação e CHANGELOG, e faça um commit pequeno e descritivo.
```
